How to Install Petra Wallet and Add the Aptos Mainnet

To install **Petra Wallet** and connect to the **Aptos Mainnet**, follow these steps:

  1. Go to the Petra Wallet website (https://petra.app/).
  2. Click "Add to Chrome" or "Add to Firefox" to install the Petra Wallet extension.
  3. Once installed, click on the Petra Wallet icon in your browser's toolbar and create a new wallet or import an existing one using your seed phrase.
  4. After setting up your wallet, click on "Networks" within Petra Wallet, and select **Aptos Mainnet**.
  5. Now, you can use your **Aptos Wallet** to send and receive APT and make donations on **MusiCoinCity**.

Once you have completed these steps, you will be able to send and receive APT on the Aptos Network using Petra Wallet. Make sure to verify recipient addresses before sending any transactions and keep your seed phrase safe.
